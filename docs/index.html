SINE

A file format containing a list of audio waves.

Disclaimer: This page is not made for phones

What is it ?

The sine file format represents a list of frequency-duration pairs to create sound. When reading a file in this format, you'll hear audio waves playing at precise frequencies and durations. You can create music, signals and so much more. All that you need is an idea for a sound and a simple command.
